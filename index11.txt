    -> What is attributes in html?
    HTML attributes are used to provide additional information about HTML elements. They allow you to modify the behavior or appearance of an element. Attributes are specified within the opening tag of an HTML element and consist of a name-value pair. The name represents the attribute, while the value provides specific information or instructions for that attribute.

    For example, you can use attributes to set the source of an image, define the target of a link, specify the dimensions of an element, or assign a class or ID for styling or JavaScript manipulation. Attributes can vary depending on the type of element they are used with.

    Some commonly used attributes include:
    - `src` for specifying the source of an image or media file
    - `href` for defining the target URL of a link
    - `class` for assigning a CSS class to an element
    - `id` for uniquely identifying an element
    - `style` for inline CSS styling
    - `alt` for providing alternative text for an image
